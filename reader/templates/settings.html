Converts digits, dates, and currency into spoken form (e.g. “2345” → “two thousand three hundred forty-five”). English/Chinese/Japanese: install wetext on Windows (or WeTextProcessing on Linux/macOS). Other languages (including Hungarian) use num2words for integers. How many coalesced utterances go into one OmniVoice.generate() call. OmniVoice is only ~0.6B params — even batch 24 often uses just a few GB and Task Manager “3D” load stays low. Check the terminal for GPU generate done: batch=… peak_alloc=… (that is the truth). Prefer nvidia-smi / Task Manager “CUDA” engine, not “3D”.
Joins consecutive same-voice sentences into one longer synth, then splits the audio back for subtitles/timeline. This usually speeds full-book export more than raising batch size alone (paper: batch 1→8 only ~30% faster).
CUDA Graph is pure PyTorch and works on native Windows (main speedup). Full Triton/hybrid needs omnivoice-triton (officially WSL2/Linux; on native Windows try pip install triton-windows then pip install omnivoice-triton). After changing this, click Reload TTS model below.
Loads a second OmniVoice model only while exporting. On an RTX 3090, two workers can overlap GPU-heavy generation with lower-load decoding. Automatically falls back to one worker when VRAM is insufficient or a worker fails.
